## Authority Tier
Court Rule (CPR 13.3)

## Currency Warning
The application MUST be made promptly. Delay is a common reason for refusal, even if there is a good defense.

## Template Type
Interlocutory Application (Set Aside)

## Structural Skeleton
1. **N244 Application Notice:** Requesting the default judgment be set aside.
2. **Evidence in Support:**
   - Explanation for the failure to file a defense (e.g., administrative error, non-service).
   - "Real Prospect of Success" (the merit of the proposed defense).
   - Demonstration of "Promptness."
3. **Draft Defense:** Should be attached to the application.

## Drafting Conventions
- **Mandatory Set Aside (CPR 13.2):** If the judgment was entered when it should not have been (e.g., defense was actually filed in time).
- **Discretionary Set Aside (CPR 13.3):** The court "may" set aside if there is a real prospect of success or other good reason.

## Fair-Use Excerpt
"The Defendant has a real prospect of successfully defending the claim because [Reason]. The application is made promptly following the Defendant becoming aware of the judgment on [Date]."

## Practical Use for Solo Advocate / Small Firm
The "rescue" application. If a client brings you a CCJ they didn't know about, this is your first step.
