# i18n/ — Constituent-Nation Language Support

UK ships with onboarding in 4 languages reflecting the UK's constituent nations:
- English (primary)
- Cymraeg (Welsh · Wales)
- Gàidhlig (Scottish Gaelic · Scotland)
- Gaeilge (Irish · Northern Ireland)

v0.1: language picker in CLI banner + 4 README variants in repo root. v0.2+ adds full UI string translation per language.

Per ADR-003: each non-English guide carries explicit "AI-assisted · native-PR welcome" header. Native speakers PR fixes via TRANSLATION_HELP_WANTED.md.
