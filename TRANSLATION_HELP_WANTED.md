# TRANSLATION_HELP_WANTED.md

AI Brain UK ships onboarding guides in 4 languages reflecting the UK's constituent nations:

| Language | Guide | Status |
|---|---|---|
| English | GETTING_STARTED.md | **Authoritative** — written by human, verified against research |
| Cymraeg (Welsh) | GETTING_STARTED_WELSH.md | **AI-assisted** — needs native speaker review |
| Gàidhlig (Scottish Gaelic) | GETTING_STARTED_SCOTTISH_GAELIC.md | **AI-assisted** — needs native speaker review |
| Gaeilge (Irish) | GETTING_STARTED_IRISH.md | **AI-assisted** — needs native speaker review |

## What we need

Native or fluent speakers of Welsh, Scottish Gaelic, or Irish to:

1. **Review** the AI-assisted translations for accuracy, natural phrasing, and domain-appropriate legal terminology
2. **Correct** any errors (grammar, vocabulary, idiom)
3. **Flag** anything that sounds inauthentic or machine-translated
4. **Suggest** better translations for UK legal terms that may not have direct equivalents

## How to contribute

1. Fork the repo: `github.com/Wolfgangrush/ai-law-firm-uk`
2. Edit the relevant GETTING_STARTED_*.md file
3. Submit a PR with a brief description of what you fixed
4. Add your name to the credits in the file if you'd like

## Credit

All native-speaker contributors will be credited in the relevant language guide and in the project README.

## For v0.2+

v0.2 aims to ship full UI string translations for all 4 languages. If you'd like to help with that effort, mention it in your PR.

---

*Per ADR-003: Each foreign jurisdiction ships with native-tongue onboarding for constituent nations/communities. AI-assisted translations carry explicit disclaimers pending native-speaker review.*
